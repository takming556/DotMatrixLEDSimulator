# DotMatrixLEDSimulator v1.1.0 by dolpast
動画ファイルを読み込んで、ドットマトリクスLED風に表示するソフトウェアです。DXライブラリを使用しています。


# 更新履歴

- v1.1.0リリース

  - 再生待機列末尾の動画再生中に乱順再生をオンにするとクラッシュする問題の解決
  - ソースコードの軽微な改良

2022年2月25日
- v1.0.2リリース
  - 全体循環がONで末端の動画を再生しているときに[次の動画へ]を押すと先頭の動画に飛ぶようにした
